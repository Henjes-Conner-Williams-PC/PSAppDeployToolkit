# Get-ADTDeferHistory

## SYNOPSIS
Get the history of deferrals in the registry for the current application.

## SYNTAX

```
Get-ADTDeferHistory [<CommonParameters>]
```

## DESCRIPTION
Get the history of deferrals in the registry for the current application.

## EXAMPLES

### EXAMPLE 1
```
Get-DeferHistory
```
